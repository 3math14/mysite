# リバーシ（オセロ）ゲーム - JavaScript版

完全にJavaScriptのみで実装されたリバーシ（オセロ）ゲームです。Pythonバックエンドやサーバーは不要で、ブラウザだけで動作します。

## 特徴

- **完全クライアントサイド**: サーバー不要、ブラウザだけで動作
- **CPU対戦と対人対戦**: 2つの対戦モードを選択可能
- **3段階の難易度**: CPU対戦では簡単・普通・難しいの3段階から選択可能
- **反転回数追跡**: 各石が反転された回数を記憶し、合計で勝敗を判定
- **反転回数表示設定**: 石ごとに反転回数を表示するか選択可能
- **美しいUI**: モダンでレスポンシブなデザイン

## 使い方

### 1. ファイルを開く

`index.html`をブラウザで開くだけです。サーバーは不要です。

```bash
# ブラウザで直接開く
# Windowsの場合
start index.html

# Macの場合
open index.html

# Linuxの場合
xdg-open index.html
```

または、ブラウザのファイルメニューから「ファイルを開く」を選択して`index.html`を開いてください。

### 2. ゲームの進行

1. **対戦モードを選択**
   - **対人対戦**: 2人で交互にプレイ
   - **CPU対戦**: コンピュータと対戦（難易度を選択可能）

2. **設定**
   - **反転回数表示**: チェックボックスで石ごとの反転回数の表示/非表示を切り替え

3. **ゲームプレイ**
   - 黒（あなた）が先手です
   - 有効な手の位置には小さな点が表示されます
   - クリックして石を置きます
   - CPU対戦モードの場合、CPUが自動的に手を打ちます

4. **勝敗の判定**
   - ゲーム終了時、獲得した石の反転回数の合計が多い方が勝ちです
   - 石の数ではなく、反転回数の合計で判定されます

## ルール

- 通常のリバーシと同じルールでプレイします
- 相手の石を自分の石で挟むと、その石を反転させることができます
- 各石は反転された回数を記憶しています
- ゲーム終了時、獲得した石の反転回数の合計が多い方が勝ちです
